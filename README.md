# Skin Lesion Classification (MILK10k)

This repository contains a university machine learning assignment done on classifying skin lesions using metadata from the **MILK10k (2025)** dataset provided by the International Skin Imaging Collaboration (ISIC).

## Dataset
- **Name**: MILK10k (2025)
- **Source**: ISIC Archive  
- **DOI**: 10.34970/6
